# chainkit

Blockchain business development toolkit

## Get started

## Develop

```bash
# Install deps
pnpm i

# Build
pnpm build

# Publish
pnpm run pub
```

## Changeset

```bash
# Init changeset
pnpm cs init

# Generate changeset
pnpm cs

# Update version
pnpm cs version

# Release
pnpm release
```
